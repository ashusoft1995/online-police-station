# 🚔 Online Police Station Criminal Management System

A comprehensive web-based system for managing criminal records, investigations, traffic accidents, and police personnel.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Spring Boot, Java 25
- **Database:** MySQL
- **Version Control:** Git & GitHub

## 🚀 Features
- Role-based access (Police Head, Detective, Traffic Police, Citizen)
- Criminal case management
- Investigation tracking
- Traffic accident reporting
- Officer management
- Citizen nominations
- Real-time notifications
